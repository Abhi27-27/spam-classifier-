# 📧 Spam Email Classifier

A machine learning project that detects whether an email or SMS message is **Spam** or **Not Spam (Ham)**.

---

## Project Overview

This project builds a spam detection model using Natural Language Processing (NLP). The model converts email text into numerical features using **TF-IDF** and classifies messages using a **Multinomial Naive Bayes** classifier.

---

## Features

- Text preprocessing and feature extraction
- TF-IDF vectorization
- Spam classification using Naive Bayes
- Model accuracy evaluation (~97%)
- Custom email prediction
